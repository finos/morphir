# Morphir IR Checker

The IR Checker tool validates your Morphir IR JSON files against the official JSON schemas.

<a href="/ir-checker" className="button button--primary button--lg">
  Open IR Checker Tool
</a>

## Features

- **Monaco Editor** - Full-featured JSON editor with syntax highlighting
- **Multiple Schema Versions** - Validate against v1, v2, v3, or v4 schemas
- **File Upload** - Upload JSON files directly
- **Sample Data** - Load sample JSON for each version
- **Detailed Errors** - See exactly where validation fails
- **Clickable Paths** - Click error paths to highlight in editor
- **Collapsible Cards** - Expand/collapse error details

## Version-Specific Checkers

For validation with integrated schema browsing, use the version-specific checkers:

- [Version 4 IR Checker](/docs/spec/ir/schemas/v4/full/schema-checker) - Draft format with wrapper objects
- [Version 3 IR Checker](/docs/spec/ir/schemas/v3/full/schema-checker) - Stable format with capitalized tags
- [Version 2 IR Checker](/docs/spec/ir/schemas/v2/full/schema-checker) - Legacy transitional format
- [Version 1 IR Checker](/docs/spec/ir/schemas/v1/full/schema-checker) - Original format with lowercase tags
